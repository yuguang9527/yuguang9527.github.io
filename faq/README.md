# FAQ 页面项目

这是一个简单的FAQ（常见问题）页面项目，使用HTML、CSS和JavaScript实现。

## 功能特点

- 可折叠的FAQ项目
- 平滑的动画效果
- 响应式设计
- 简洁的用户界面

## 文件结构

- `index.html` - 主页面HTML文件
- `styles.css` - 样式表文件
- `script.js` - JavaScript交互脚本
- `README.md` - 项目说明文档

## 使用方法

1. 克隆此仓库
2. 在浏览器中打开 `index.html` 文件
3. 点击问题标题即可展开/折叠答案

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
